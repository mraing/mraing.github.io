计算机组成与体系结构
2020年10月18日 1.2k 字 大概 4 分钟 软件设计师 软考笔记

  1. 阶码决定数表示的范围，尾数的长度决定数表示的精度。

  2. 两个浮点数相加，首先要统一阶码。

  3. ±0 编码相同的是补码和移码。

  4. 逻辑加：
    A或B只要有一个为1，其逻辑加的结果就为1；
    只有当两者都为0时，逻辑加的结果才为0。

    逻辑异或:
    0同0 异或 为 0，
    0同1 异或 为 1，
    1同0 异或 为 1，
    1同1 异或 为 0

  5. 补码可以简化计算机运算部件的设计。

  6. 程序计数器 (PC) 是专门寄存器，具有寄存信息和计数两种功能，又称为指令计数器。

  7. 程序被加载到内存后开始运行，当CPU执行一条指令时，先把它从内存储器取到缓冲寄存器DR中，再送入IR暂存，指令译码器根据IR的内容产生各种微操作指令，控制其他的组成部件工作，完成所需的功能。

  8. 指令寄存器用来存放当前的指令，对用户是完全透明的。

  9. CPU 中译码器的主要作用是指令译码。

  10. 在 CPU 中，控制器不仅要保证指令的正确执行，还要能够处理异常事件。

  11. 在 CPU 中，为 ALU 执行算数逻辑运算提供数据并暂存运算结果的是 累加寄存器。

  12. 属于 CPU 中算数逻辑单元的是加法器。

        运算器：
    * 算数逻辑单元 ALU
    * 累加寄存器器
    * 数据缓冲寄存器
    * 状态条件缓存器

    控制器：
    * 程序计数器 PC
    * 指令寄存器 IR
    * 指令译码器
    * 时序部件
  13. 在 CPU 执行指令的过程中，会自动修改PC 的内容，PC 是指令计数器，用来执行即将要执行的下一条指令。
         * 指令寄存器IR: 存放即将要执行的指令。
     * 指令译码器ID: 对指令中的操作码和字段进行分析和解释。
     * 地址寄存器AR: 保存当前 CPU 要访问的内存单元或 I/O设备的地址。

  14. 计算器执行指令过程中，需要由CPU的控制器产生每条指令的操作信号并送往相应的部件进行处理，并完成指定的操作。
    CPU 的运算器只完成运算，而控制器控制着整个 CPU 的工作。

  15. Flynn 分类法

         * SISD 传统的串行计算机。
     * SIMD 一个指令流处理多个数据流。
     * MISD 多个指令流处理单个数据流。
     * MIMD 同时执行多个指令，并处理个不同的数据。

  16. 流水线的吞吐率是指单位时间流水线输出的结果的数目。因此流水线的吞吐率为一个流水级时间的倒数，即最长流水级的倒数。

  1. 相连存储器是按内容访问的存储器。

  2. 虚拟存储器由主存和辅存构成。

  3. 计算机中CPU 对访问速度最快的是： 通用寄存器 > Cache > 内存 > 硬盘 。

  4. 计算机中的主存主要有 DRAM 构成的。

         * DRAM 动态随机存取存储器
     * SRAM 静态随机存取存储器
     * Cache 高速缓存
     * EEPROM 电可擦可编程只读存储器

  5. RAM 断电会丢失，ROM 断电不会丢失。

  6. 高速缓存Cache的存储系统的平均存取时间为

    TA = HTA1 + (1 - H) TA2

    • 平均存取时间：TA
    • 命中率：H
    • 高速缓存存取速度：TA1
    • 主存存取速度：TA2

  7. Cache与主存的地址映像需要专门的硬件自动完成，使用硬件来处理具有更高的转换效率。

  8. CPU 中数据总线的宽度会影响系统的运算速度。

  9. 处理机主要由处理器，存储器，总线构成。其中总线包括数据总线，地址总线，控制总线。

  10. 在计算机系统中采用总线构造，便于实现系统的积木化构造，同时可以减少信息线传输的数量。

  11. 软件的可靠性，可用性，可维护性，可伸缩性。

         * 软件的可靠性是指一个系统在给定时间间隔内和给定条件下无失效运行的概率。
     * 软件的可用性是指软件在特定使用环境下为特定用户用于特定用途时所具有的有效性。
     * 软件的可维护性是指与软件维护的难易程度相关的一组软件属性。
     * 软件的可伸缩性是指是否可以通过运行更多的实例或者采用分布式处理来支持更多的用户。

  12. 指令系统中采用不同的寻址方式目的是扩大寻址空间，并提高编程灵活性。

流水线

流水线是指在程序执行时，多条指令重叠进行操作的一种准并行处理实现技术。

各种部件处理是针对不同指令而言的，它们可同时为多条指令的不同部分进行工作，以提高各部件的利用率和指令的平均速度。
