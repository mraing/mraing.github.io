docsify 是一个文档站点生成器。

使用起来非常简单。也是采用 Markdown 语法来编写文档。

但它与 Hexo 不同的是，Hexo 是编译完成之后再展示，会把源文件转换成 html 文件。

docsify 是在展示时再进行实时编译，这样做的好处是.md源文件不会变成 .html 文件。
